# 🩺 Medibot - AI Health Assistant

**Medibot** is an AI-powered chatbot designed to answer medical questions. It is built on the **DigitalOcean AI Agent** platform, leveraging the advanced reasoning capabilities of OpenAI's latest models (like GPT-5) and a vast, curated knowledge base to provide accurate, contextual, and reliable healthcare insights.

---

## 📌 Description

This application connects a user-friendly Streamlit frontend to a powerful, managed AI backend hosted on DigitalOcean.

The backend handles all the complex "Retrieval-Augmented Generation" (RAG) processes. When a user asks a question, the DigitalOcean agent intelligently retrieves the most relevant medical information from its vast knowledge base and then uses an advanced GPT model to formulate a safe, accurate, and easy-to-understand answer.

---

## 🚀 Features

-   💬 **Simple Interface:** A clean and intuitive chat interface built with Streamlit.
-   🧠 **Advanced Reasoning:** Powered by OpenAI's state-of-the-art GPT models (e.g., GPT-5/GPT-4o).
-   📚 **Vast Knowledge Base:** Taps into a large, managed database of medical information for high-quality, contextual answers.
-   🛡️ **Managed RAG:** The entire retrieval and generation process is managed by the DigitalOcean AI platform, ensuring reliable and accurate responses.
-   ⚡ **Scalable & Fast:** Built on DigitalOcean's robust infrastructure for a fast and scalable user experience.

---

## 🛠️ Tech Stack

-   **Frontend:** Streamlit
-   **Backend:** Python
-   **AI Platform:** DigitalOcean AI Agents
-   **LLM:** OpenAI (GPT-5 / GPT-4o)
-   **Knowledge Base:** DigitalOcean-managed Vector Database
-   **Deployment:** Streamlit (Frontend) & DigitalOcean (AI Agent Backend)
